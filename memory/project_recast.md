Standalone product owned by user, sub-product under Nexonauts brand.

State as of 2026-05-23:
- Desktop app fully complete for Windows; testing only
- Mac/Linux support being added
- Cloud "Loom-like" sharing feature in progress in parallel (NOT required for v1 launch)
- ~2-3 word-of-mouth users trying it out; no official public launch yet
- User has been sharing progress on Twitter but not announced a release
- **Now has its own deployed marketing site at `recast.nexonauts.com`** — the internal `/recast` route in the Nexonauts repo is obsolete and should be removed in the next design pass. Link out to the subdomain instead.

**Why this matters:** Closest product to shippable in his portfolio. Strong candidate for first public launch under Nexonauts brand. Risk: user is building cloud sharing in parallel with finishing desktop — this is the same parallel-half-built pattern that produced the half-baked Nexonauts main repo.

**How to apply:** Push for shipping Windows desktop publicly first, treating cloud as a separate v2 marketing moment. Resist scope additions to v1.
